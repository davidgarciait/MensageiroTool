Para executar isto é preciso ter o Node instalado


1- Editar o ficheiro server.js
	
	- Editar a linha 9 e colocar a pasta dos vossos ficheiros do mensageiro
		(Dica: Ir ao mensageiro -> Ferramentas -> Registo de conversas -> Copiar o endereço da pasta)
	
2 - Editar o ficheiro index.html
	- Editar a linha 214 para colocar o vosso nome de forma a que as vossas mensagens se destaquem em relação ao outro contacto.
	
3- Executar o node
	- Abrir o terminal na raiz da pasta
	- Executar o codigo "node server.js"
	
4- Abrir o ficheiro index.html no browser
